## Команды

Все команды трехбайтовые

MQTT -> GPIO

| команда               | описание                   |
|-----------------------|----------------------------|
| 00 (00..32) хх        | выключить порт (00..32)    |
| 01 (00..32) хх        | включить порт (00..32)     |
| 02 (00..03) (0...100) | установить PWM в процентах |
| 03 byte0 byte1        | команда DALI               |
